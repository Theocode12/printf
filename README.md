## Setup
* Clone repository with:
```
git clone https://github.com/Theocode12/printf
```
* cd into repo:
```
$ cd ./path/printf
```
* Compile code with:
```
$ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c
```
* Check man page with
```
$ man ./man_3_printf.man/
```
